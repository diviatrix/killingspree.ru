Роли и команды

У игрока есть базовый трек по ролям - Default Track.
Внутри трека есть две роли: Default, Player.

Default

Эта роль выдаётся игроку по умолчанию при регистрации.
Игрок с ролью Default не может строить, ломать блоки и т.д.

Доступные команды:

/register <пароль> <пароль> - регистрация
/login <пароль> - вход
/logout - выход
/unregister <пароль> - удаление аккаунта
/changepassword <старый пароль> <новый пароль> <новый пароль> - смена пароля
/spawn - телепорт на спавн
/warp - список точек телепорта
/warp <точка> - телепорт на точку

Player


Чтобы получить роль Player достаточно просто кликнуть RMB на табличку.
Игрок с ролью Player может полностью использовать всю включенную для игрока функциональность.
Так же, роль Player включает в себя доступы роли Default

Антигрифинг и Приватная территория

Создание приватного участка

Для создания своей территории (приватный участок) достаточно поставить сундук на свобдной территории
Так же, создать приватный участок можно держа в руках Золотую Лопату и кликнув по углам, после чего в чате появится подсказка 

/claim - создание приватного участка
/unclaim - удаление приватного участка
Ресурсы для лопаты можно купить в админ магазине на /spawn
Так же, при использовании лопаты визуально видны границы вашего участка



Расширение участка

На старте игроку выдается 1000 блоков для привата
Количество блоков для привата увеличивается со временем игры

Для расширения участка нужно взять в руку Золотую Лопату, встать на свой участок и использовать команду /expandclaim
/expandclaim <количество> <направление> - расширение участка
/expandclaim 100 west - Для расширения участка на 100 блоков на запад
Направление можно посмотреть нажав F3 или на карте
Там же на карте можно увидеть границы своего участка

Все доступные команды:

/AbandonClaim - удаление приватного участка
/ClaimExplosions - разрешение взрывов на вашем участке
/Trust /t <игрок> - добавление игрока в список доверенных
/UnTrust /ut <игрок> - удаление игрока из списка доверенных
/AccessTrust /at <игрок> - добавление игрока в список доступа

/TrustList - список доверенных игроков
/SubdivideClaims /sc - разделение участка на подучастки
/RestrictSubclaim /rs - ограничение доступа к подучастку
/BasicClaims - включение/выключение базовых приватов
/PermissionTrust /pt <игрок> - добавление игрока в список доступа к командам
/Untrust All - удаление всех доверенных игроков
/AbandonAllClaims - удаление всех приватных участков
/BuyClaimBlocks <количество> - покупка блоков для привата
/SellClaimBlocks <количество> - продажа блоков для привата
/GivePet <игрок> - передача питомца

Навыки и прокачка

На сервере включен плагин AuraSkills.

Плагин позволяет игрокам прокачивать свои навыки и умения.
Прокачка навыков происходит за счет опыта, который игрок получает за выполнение различных действий.
Все навыки и умения можно посмотреть в меню навыков, которое открывается командой /skill


Доступные команды:

/skill - открытие меню навыков
/skill <claimitems> - собрать все предметы
/skill <help> - помощь по навыкам
/skill <lang> <ru> - выбор языка
/skill <rank> - просмотр уровня навыков
/skill sources <навык> - просмотр источников опыта, например /skill sources agility
/archery - просмотр панели навыков стрельбы из лука
/agility - просмотр панели навыков ловкости
/alchemy - просмотр панели навыков алхимии
/defense - просмотр панели навыков защиты
/excavation - просмотр панели навыков копания
/enchanting - просмотр панели навыков зачарования
/farming - просмотр панели навыков фермерства
/fishing - просмотр панели навыков рыбалки
/fighting - просмотр панели навыков б
/foraging - просмотр панели навыков сбора
/mining - просмотр панели навыков горного дела

/stats - просмотр статистики
/abtoggle - включение/выключение ActionBar
/skilltop <навык> - просмотр топа по навыку
/skilltop - просмотр топа по всем навыкам
/mana - просмотр маны
